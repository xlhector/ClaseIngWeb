Comando Descripción
Git Init Inicializa la carpeta que quedará como repositorio.
Git add -A Coloca los archivos en el area de STAGING.
Git commit -m "Palabras" Realiza un commit y aplica los cambios.
Git log Permite ver todos los commit realizados anteriormente.
Git push origin Envía los datos al servidor.
